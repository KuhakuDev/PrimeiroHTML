Funcionamento das Tags H1, H2... H6.


Uma das primeiras e mais importantes Tags que devem ser aprendidas.
A Tag 'h...' é normalmente utilizada com as númerações de 1 à 6.
Esta Tag é utilizada para demarcar Titulos e Subtitulos e determina o tamanho do texto, sendo a Tag h1 o texto com maior tamanho e a Tag h6 a de menor tamanho.
Sua estrutura é realizada da seguinte forma:
< h1 > 'Titulo Aqui' < /h1 >

Utilizando < p >

A Tag < p > e comummente utilizada nos projetos. A mesma e denominada como paragráfo.
Ou seja, e utilizada para que se escrevam paragráfos, onde é aberta a tag de forma:
< p > 'Paragráfo escrito aqui' < /p >
Desta forma podemos identificar que o paragráfo e fechado utilizando um fechamento de tag com < / p > .

Utilizando < mark >

O comando < mark > é comummente utilizado para realizar marcações em palavras ou trechos.
Seu trabalho e destacar a parte selecionada como uma marcação, desta forma.
A tag é estruturada da seguinte forma:
< mark > 'texto aqui' < /mark >

Utilizando < small >

A tag < small > é utilizada para definir um texto com tamanho menor, como em informações de direitos autorais ou comentários em textos, imagens, etc.
Sua estrutura é realizada da seguinte forma:
< small > 'texto aqui' < /small >

Utilizando < i >

A Tag < i > é muito utilizada em texto, como uma função muito simples, porém de muita utilidade.
Utilizada para mudança de formatação de texto, a Tag < i > é utilizada para deixar um texto, frase ou palavra em ITALICO.
Sua contrução é realizada da seguinte forma:
< i > 'frase aqui' < /i >

Utilizando < u >

A Tag < u > é utilidade de como Tag de formatação de texto, assim como outras Tags similares.
Sua função é realizar a sublinhação de palavras. Que pode ser utilizada de diversas formas.
Sua construção é feita da seguinte forma:
< u > 'Texto aqui' < /u >

Utilizando < strong >

A Tag < strong > é utilizada como fomratação de texto.
Tem como função a realização da formatação da texto, frases ou palavras para negrito.
Sua estrutura é realizada da seguinte forma:
< strong > 'Frase Aqui' < /strong >
